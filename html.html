HTML

What is HTML?

HTML (HyperText Markup Language) is a coding language that acts as the skeleton of the website, setting the structure of the content and web pages.

How is HTML written?

HTMl is written using tags. There is an opening tag < tagname > and a closing tag, < /tagname >. The content goes in between the tags.

Common Tags

  • < html > - Root of HTML Document
  • < head > - Info about HTML Document
  • < body > - Document's Body
  • < h1 > < h2 > < h3 >,....,< h6 > - "Header" Tags
  • < p > - Paragraph tags
  • < a > - Stands for anchor, it is used for links. It also includes an attribute, which adds additional information to the element
  • < img > - For images and the src attribute is also included
  • < div > - Block section in document
  • < span > - Incline section in document
