CSS

What is CSS?

CSS(Cascading Style Sheets) is a style sheet language used to describe the presentation and visual formatting of a document written in HTML (Hypertext Markup Language) or XML. Essentially, CSS determines the appearance of the written code, allowing creativity and originality via fonts, colors, backgrounds, animations, etc.

How to use CSS?

CSS utilizes two main methods: in-line and external. Internal is done within the HTML code using a style tag in the head section of an HTML document, hence the name "in-line". External is done in a separate .css file and linked to the HTML document using a link tag in the head section.

This structure of CSS in an external file includes: selector, property, and value.

Selector refers to the element that you are customizing. It includes headings, paragraphs, body of HTML document, divs, classes, and IDS. The property defines what you are changing - background, size, font-family, etc. - while value is what you are changing it to.

How to link CSS to HTML?

To link the CSS properties to the HTML code, there must be a link tag connecting the .css file to the HTML document.

This is done with the link tag, followed by the defining relationship and type, and ending with href - the .css file that is being edited.

Class and ID Attributes

Any HTML tag can have class(es) and/or an id attribute. This means putting an element(s) under a specific group to be altered in CSS. Classes can have multiple elements and be used multiple times, meaning they are not unique. To start the class code in CSS, you start with a period. However, ID's can only be used once per element, making them unique. The start for the ID code is (#). Class and ID are not their own tags, they must go after the opening part of a tag (e.g. body, heading, paragraph).
