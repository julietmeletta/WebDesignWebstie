'apple' is of type String, meaning it is a string of characters. JavaScript has 5 primitive types: Number, String, Boolean, Null, and Undefined. Null and Undefined, while sounding similar, have distinct meanings. Null is the purposeful absence of data, Undefined is something that will be given data, but we haven't given the variable that data yet. That's all you need to know about variables for now!
